# claude-skills

- **[qa-loop](qa-loop/SKILL.MD)** — Autonomous QA engineer that verifies an implementation via real execution (Playwright, screenshots, console/network checks) and runs a bounded repair loop until it passes.
- **[roast](roast/SKILL.MD)** — Stress-tests a business/product idea through an adversarial five-persona council and issues a GREEN LIGHT / RESHAPE / KILL verdict.
- **[production-grade](production-grade/SKILL.MD)** — Principal-engineer persona that generates production-ready code: secure, leak-free, observable, performant, and conceptually clear.
- **[spec-cleanup](spec-cleanup/SKILL.MD)** — Final-pass cleanup that rewrites a multi-draft spec/design doc into a clean, standalone specification.
- **[bank-statement-analyzer](finance/personal_finance/bank-statement-analyzer/SKILL.MD)** — Extracts, categorizes, and analyzes bank statement transactions into a privacy-safe cash flow report.
- **[credit-card-statement-analyzer](finance/personal_finance/credit-card-statement-analyzer/SKILL.MD)** — Analyzes credit card statements for spending, utilization, fees, rewards, and payment behavior.
- **[expenditure-synthesizer](finance/personal_finance/expenditure-synthesizer/SKILL.MD)** — Combines bank and credit card analyses into one reconciled, double-count-free financial report.
